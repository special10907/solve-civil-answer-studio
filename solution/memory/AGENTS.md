# Memory/ — Your Persistent Memory

This folder holds everything that persists across sessions. Treat it as your long-term brain.

## Files

- **Memory/BOOTSTRAP.md** — setup guide, present only when (re)initialization is needed. Follow it, then delete it.
- **Memory/SOUL.md** — your soul: who you are and how you operate, in your own voice
- **Memory/IDENTITY.md** — your character card: name, role, vibe, capabilities
- **Memory/USER.md** — information about the person you're helping
- **Memory/MEMORY.md** — curated long-term memory (distilled insights, not raw logs)

These files are auto-injected into your context at session start. You don't need to read them manually.

## Daily Notes

In addition to the auto-injected files, check `Memory/journal/YYYY-MM-DD.md` (today + yesterday) for recent context. These are NOT auto-injected — you need to read them yourself.

Journal files are append-only — always add new entries at the end. Never modify or overwrite existing content.

## Write It Down

You wake up fresh each session. These files are your continuity.

- If you want to remember something, **write it to a file**. "Mental notes" don't survive sessions.
- When someone says "remember this" → update `Memory/journal/YYYY-MM-DD.md` or the relevant file.
- When you learn a lesson → update the relevant file so future-you doesn't repeat it.

Over time, review daily files and distill what's worth keeping into `Memory/MEMORY.md`.

## Make It Yours

This is a starting point. Add your own conventions and rules as you figure out what works.
